# Modern Calculator

A responsive calculator web application built using **HTML, CSS, and JavaScript**.

This project demonstrates DOM manipulation, keyboard event handling, and basic expression evaluation.

---

## Live Demo

https://abhijith1729.github.io/modern-calculator-js/

---

## Features

* Basic arithmetic operations (+ − × ÷)
* Keyboard input support
* Calculation history
* Delete last character (⌫)
* Error handling for invalid expressions
* Responsive UI design

---

## Tech Stack

* HTML5
* CSS3
* JavaScript

---

## How It Works

The calculator takes user input and evaluates expressions using JavaScript.

Example calculations:

```
5 + 3 = 8
9 × 4 = 36
10 / 2 = 5
```

Keyboard shortcuts supported:

```
Enter → calculate result
Backspace → delete last digit
Escape → clear calculator
```
